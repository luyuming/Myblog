# Myblog
这是一个用go实现的简易博客<br>
- web框架：gin
- 存储：mysql+gorm
- 点赞: redis
- 注册邮件通知：rabbitmq

myblog只具备简单的注册、登录、浏览、发表、评论和点赞功能<br>
